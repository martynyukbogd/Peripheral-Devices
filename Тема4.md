# Тема 4: Слоти та карти розширення (Expansion Slots & Cards)

## 1. Огляд та характеристики слотів розширення

**Слот розширення (Expansion Slot)** — це роз'єм на материнській платі, призначений для встановлення додаткових плат (карт розширення), які додають нові функції комп'ютеру або покращують наявні.



### Основний стандарт: PCI Express (PCIe)
На сучасних материнських платах основним стандартом є **PCI Express (Peripheral Component Interconnect Express)**. Це послідовний інтерфейс з високою пропускною здатністю.

#### Ключові характеристики PCIe:
1.  **Лінії (Lanes):** Слоти розрізняються за фізичним розміром та кількістю ліній передачі даних.
    * **x1:** Найменший, використовується для простих пристроїв (Wi-Fi, аудіо).
    * **x4:** Середній розмір, часто використовується для SSD-адаптерів або контролерів захоплення відео.
    * **x8:** Рідше зустрічається окремо, часто це фізичний слот x16 з розпайкою тільки половини контактів.
    * **x16:** Найдовший, використовується для відеокарт.

2.  **Покоління (Generations) та швидкість:**
    | Версія PCIe | Швидкість на одну лінію (GB/s) | Пропускна здатність x16 (GB/s) |
    | :--- | :--- | :--- |
    | 3.0 | ~0.985 GB/s | ~15.75 GB/s |
    | 4.0 | ~1.969 GB/s | ~31.51 GB/s |
    | 5.0 | ~3.938 GB/s | ~63.02 GB/s |

3.  **Сумісність:** Меншу карту (наприклад, x1) можна вставити у більший слот (наприклад, x16), і вона працюватиме.

## 2. Карти розширення на інтерфейсі PCI-e (8 прикладів)

Нижче наведено 8 типів карт розширення для різних потреб та слотів.

### 1. Дискретна відеокарта (GPU)
* **Тип слоту:** PCIe x16
* **Призначення:** Обробка графіки, ігри, рендеринг відео, 3D-моделювання. Це найпотужніші та найбільші карти розширення.
* ![Відеокарта](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTqw9uYzgoTcAh-7v_3dxQioGbyfSxbNhl5Qw&s)

### 2. Мережевий адаптер Wi-Fi + Bluetooth
* **Тип слоту:** PCIe x1
* **Призначення:** Додавання бездротового зв'язку до настільного ПК. Зазвичай мають зовнішні антени для кращого сигналу.
* ![Wi-Fi карта](https://click.ua/content/shop/products/126895/photos/besprovodnoy-adapter-fenvi-fvminusac1200-800x800-8c09.jpg)

### 3. Адаптер M.2 NVMe to PCIe
* **Тип слоту:** PCIe x4
* **Призначення:** Дозволяє підключити швидкісний SSD накопичувач формату M.2 до материнської плати, яка не має вільного M.2 роз'єму, або для створення RAID-масиву з SSD.
* ![NVMe адаптер](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c2/M.2_adapter.jpg/320px-M.2_adapter.jpg)

### 4. Звукова карта (Audio Card)
* **Тип слоту:** PCIe x1
* **Призначення:** Забезпечення високоякісного звуку (Hi-Fi), підтримка об'ємного звуку 5.1/7.1, зниження затримок при запису музики.
* ![Звукова карта](https://upload.wikimedia.org/wikipedia/commons/thumb/7/77/Sound_Blaster_Live%21_5.1.jpg/320px-Sound_Blaster_Live%21_5.1.jpg)

### 5. Мережева карта Ethernet (10 Gigabit LAN)
* **Тип слоту:** PCIe x4 або x8
* **Призначення:** Забезпечення надшвидкого дротового інтернет-з'єднання (швидше за стандартні 1 Гбіт/с порти на платі) для серверів або робочих станцій.
* ![Мережева карта](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9e/Network_card.jpg/320px-Network_card.jpg)

### 6. Карта захоплення відео (Capture Card)
* **Тип слоту:** PCIe x4
* **Призначення:** Запис відеосигналу з зовнішніх джерел (ігрові консолі, камери) на комп'ютер. Використовується стрімерами.

### 7. Контролер портів USB (USB Expansion Card)
* **Тип слоту:** PCIe x1
* **Призначення:** Додавання додаткових портів USB (Type-A або Type-C), якщо вбудованих не вистачає або вони застарілі.

### 8. RAID-контролер
* **Тип слоту:** PCIe x8 або x16
* **Призначення:** Апаратне керування масивом з багатьох жорстких дисків (HDD) для серверів. Забезпечує надійність збереження даних та швидкість.
